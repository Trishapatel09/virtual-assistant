#  Virtual Voice Assistant (Python)

## About
This is a basic voice-controlled assistant built using Python.  
It takes voice input from the microphone, converts it into text, and performs simple tasks like opening websites, searching Google, and fetching information from Wikipedia.

## Features
 Voice input using microphone  
 Text-to-speech output  
 Open websites (Google, Instagram, ChatGPT)  
 Wikipedia search  
 Google search for unknown queries  
 Basic responses (hello, how are you, help)

## Technologies Used
 Python  
 pyttsx3  
 SpeechRecognition  
 Wikipedia  
 webbrowser  
 os  

## How to Run

### 1. Install dependencies
pip install pyttsx3 SpeechRecognition wikipedia pyaudio

### 2. Run the program
python virtualassistent.py

## Example Commands
 open google  
 open instagram  
 chatgpt  
 search python on wikipedia  
  how are you
  
## Limitations
 Speech recognition may not always be accurate  
 Works on basic keyword matching  
 Requires internet connection
